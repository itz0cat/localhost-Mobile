Hosting a Minecraft Server on Termux & AnLinux

Step 1 - Setup

1. Install Termux from GitHub or F-Droid.

2. Install AnLinux from the Google Play Store.

3. Open AnLinux → Dashboard → Select Ubuntu.

4. Copy the provided command and press Launch.

5. Open Termux, paste the command, and wait for installation to complete.

./start-ubuntu.sh

Step 2 - Install Required Packages

apt update && apt install -y nano openjdk-17-jre

Step 3 - Download the Minecraft Server

Select your Minecraft version:

wget -O minecraft_server.jar https://api.papermc.io/v2/projects/paper/versions/1.21.4/builds/130/downloads/paper-1.21.4-130.jar

Step 4 - Make the File Executable

chmod +x minecraft_server.jar

Step 5 - Run the Server

Replace (RAM) with your desired RAM allocation.

java -Xmx2G -Xms2G -jar minecraft_server.jar nogui
