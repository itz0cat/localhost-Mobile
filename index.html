Minecraft Server Setup on Android

Step 1 - Setup

Run the following command:

./start-ubuntu.sh

Step 2 - Install Required Packages

Run this combined command:

apt update && apt-get clean && apt-get install -y build-essential && apt-get install software-properties-common && add-apt-repository ppa:openjdk-r/ppa && apt-get update && apt-get install openjdk-8-jre && apt-get install openjdk-21-jdk

Step 3 - Download Minecraft Server

wget -O minecraft_server.jar (your_server_file_link)

Step 4 - Make the File Executable

chmod +x minecraft_server.jar

Step 5 - Run the Server

java -Xmx(MAX_RAM) -Xms(MIN_RAM) -jar minecraft_server.jar nogui

Step 6 - Accept EULA

nano eula.txt

Step 7 - Restart Server

java -Xmx(MAX_RAM) -Xms(MIN_RAM) -jar minecraft_server.jar nogui
